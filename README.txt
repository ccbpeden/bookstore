# Bookstore

## Description

A barebones bookstore application generated using Drupal 7.

## Epicodus Drupal Week 1 Project 4/21/2017

## By Charles Peden

## Prerequisites

You will need the following software properly installed on your computer.

* [MAMP](https://www.mamp.info/en/)
* [Composer](https://getcomposer.org/) (with NPM)


## Installation

* `git clone https://github.com/ccbpeden/bookstore.git` this repository
* `cd bookstore`
* activate mamp and import the app database from bookstore/sites/db-backup/bookstore.sql
* in mamp, add a user to the bookstore database with username 'bookstore' and password 'bookstore'
* in mamp, set the document root to the top level of the bookstore folder.
* navigate to http://localhost:80/
